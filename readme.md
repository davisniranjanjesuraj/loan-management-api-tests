#  Loan Management System – API Testing Framework

##  Overview
This project is an **API Testing Framework** for a Loan Management System, designed to validate complete backend loan workflows **without any UI dependency**.  
It focuses on functional, negative, and workflow-based API testing using Postman and Newman.

The framework simulates real-world banking scenarios such as:
- Loan creation
- Loan approval
- Loan disbursement
- Repayment processing
- Invalid request handling

---

##  Tech Stack
- **Postman** – API test creation
- **Newman** – CLI execution of Postman collections
- **REST APIs**
- **JSON**
- **json-server** – Mock backend
- **GitHub Actions** – CI execution (optional)
